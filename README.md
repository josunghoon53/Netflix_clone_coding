## 프로젝트 소개
영화,드라마 등 영상 스트리밍 사이트인  NETFLIX를 클론 코딩하였습니다.</br>
클론코딩을 함으로써 비동기처리를 통한 API호출, 컴포넌트의 재사용 , SASS 사용 등</br> 
REACT를 다양하게 활용함으로서 기술 숙련도를 높이고자 하였습니다.</br>
